# Grocery Store Website

A responsive grocery store website built with React, Tailwind CSS, React Router DOM, Swiper.js, and React Icons.

## Features

* Responsive design for mobile, tablet, and desktop
* Product showcase sections
* Smooth navigation with React Router DOM
* Interactive sliders using Swiper.js
* Modern UI with Tailwind CSS
* Beautiful icons with React Icons

## Technologies Used

* React.js
* React Router DOM
* Tailwind CSS
* Swiper.js
* React Icons
* JavaScript

## Installation

bash
npm install
npm run dev
